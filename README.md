Overview 

This project implement a complete Devops Workflow
-Python flask application --- build in local host and created a virtual env using
python -m venv venv
- Installed Dependecies
-Docker containerization  --- Build a Docker Image for the apploication 
-Docker Hub Image Push --- Pushed the Image into Docker Hub
-GitHub Actions CI/CD --- Developed a CI/CD pipeline by integrating the Dockerhub
-Trivy vulnerability scanning --- For application scanning
-Monitoring -- A File has been added to monitor the application
